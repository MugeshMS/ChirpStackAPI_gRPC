# ChirpStack Gateway gRPC Dashboard

This project implements a Gateway Management Dashboard using ChirpStack gRPC APIs.

## Features
- List gateways
- View gateway details
- Online / Offline status
- Last seen timestamp
- RX / TX packet metrics
- Frequency-wise metrics
- Data-rate-wise metrics
- Charts using Chart.js
- Spring Boot + Thymeleaf UI

## Tech Stack
- Java
- Spring Boot
- gRPC
- ChirpStack API
- Thymeleaf
- Chart.js

## Note
API keys and tenant IDs are managed securely and not committed to the repository.
